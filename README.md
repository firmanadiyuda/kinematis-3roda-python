# Kinematis 3 Roda dengan Python

---

## Tentang
Program robot kinematis 3 roda.

## Prequisites
Install 'smbus-cffi':
```bash
sudo apt-get install build-essential libi2c-dev i2c-tools python-dev libffi-dev
```

Install driver PCA0695 melalui pip:
```bash
pip install PCA9685-driver
```

## Run
```bash
python start.py
```
